<h2><a href="https://leetcode.com/problems/find-the-difference">Find the Difference</a></h2>
<img src="https://img.shields.io/badge/Difficulty-Easy-brightgreen" alt="Difficulty: Easy" />
<hr>

<p>Given two strings <code>s</code> and <code>t</code>, where <code>t</code> is generated by random shuffling <code>s</code> and then adding one more letter at a random position, return the letter that was added.</p>

<p><strong>Example 1:</strong></p>
<pre>
Input: s = "abcd", t = "abcde"
Output: "e"
Explanation: 'e' was added.
</pre>

<p><strong>Example 2:</strong></p>
<pre>
Input: s = "", t = "y"
Output: "y"
</pre>

<p><strong>Example 3:</strong></p>
<pre>
Input: s = "a", t = "aa"
Output: "a"
</pre>

<p><strong>Constraints:</strong></p>
<ul>
  <li>0 ≤ s.length ≤ 10<sup>4</sup></li>
  <li>t.length = s.length + 1</li>
  <li><code>s</code> and <code>t</code> consist of only lowercase English letters.</li>
</ul>

<hr>

<h3>Solution</h3>

<p><strong>Approach:</strong><br>
Use a dictionary to count characters in <code>s</code>, then iterate over <code>t</code> to find the character not in <code>s</code> or that exceeds its count.  
Alternatively, XOR all characters in <code>s</code> and <code>t</code> to find the extra character efficiently.
</p>

<p><strong>Complexity:</strong></p>
<ul>
  <li>Time: <code>O(n)</code></li>
  <li>Space: <code>O(1)</code> (or O(n) if using a dictionary)</li>
</ul>
